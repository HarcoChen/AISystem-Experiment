# 实验环境说明

## 硬件环境

### CPU

- 型号：Intel(R) Xeon(R) Gold 6226R CPU @ 2.90GHz
- 主频：2.90 GHz
- 物理核心数：单路 16 核
- 逻辑线程数：单路 32 线程

说明：从系统信息看，当前机器为多路服务器环境，`processor : 63` 表明系统可见 64 个逻辑 CPU。

### GPU

- GPU 型号：NVIDIA GeForce RTX 3090
- GPU 数量：8
- 单卡显存：24 GB
- 驱动版本：570.133.20
- CUDA 版本：12.8

`nvidia-smi` 信息表明该服务器共有 8 张 RTX 3090，可满足单机多卡分布式训练实验要求。

## 软件环境

- 操作系统：Linux
- Python：已使用 Conda 环境运行实验
- PyTorch：用于实现 `torch.distributed` 与 DDP 分布式训练
- CUDA：12.8
- NVIDIA Driver：570.133.20

## 环境分析说明

本实验使用的硬件配置较高，GPU 资源充足，适合进行 1 卡、2 卡和 4 卡分布式训练对比实验。CPU 端具备较多核心和线程，理论上能够支持 CIFAR-10 数据加载与图像增强任务，因此实验中若出现通信占比波动较大或结果与理论趋势不完全一致，更可能与通信时间近似测量方法、CUDA 预热、缓存状态以及计算与通信重叠等因素有关，而不一定完全由 CPU 性能不足导致。
